# API Integrations

## GitHub

Ingests PRs, issues, and commits via the GitHub REST API.

## Slack

Ingests channel messages relevant to project activity.

## Jira

Ingests issues and comments.

## Google Drive

Ingests shared documents for context extraction.

## LLM Providers

Claude and Gemini are used for fact extraction and chat/briefing generation. Prompt templates live in [prompts/](../prompts/).
